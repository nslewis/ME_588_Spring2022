# Foobar

Foobar is a Python library for dealing with word pluralization.

## Installation

Use FSM_Final.ino to run the robots. 

```bash
pip install foobar
```

## Usage

```python
import foobar

# returns 'words'
foobar.pluralize('word')

# returns 'geese'
foobar.pluralize('goose')

# returns 'phenomenon'
foobar.singularize('phenomena')
```
